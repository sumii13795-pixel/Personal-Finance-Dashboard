# Personal-Finance-Dashboard
 Personal Finance Dashboard in Excel to track income, expenses, savings, and financial insights using interactive charts and data analysis.
📌 Project Overview
This project is an advanced Excel-based Personal Finance Dashboard designed to track income, expenses, savings, and financial trends.
It helps users make better financial decisions using interactive dashboards and automated calculations.

🚀 Features
✔ Income & Expense Tracking
✔ Monthly Budget Analysis
✔ Category-wise Expense Breakdown
✔ Savings Tracking
✔ Interactive Dashboard with Charts
✔ Automated Calculations using Excel formulas

📊 Dashboard Insights
Total Income vs Expenses
Savings Overview
Expense Categories (Food, Rent, Travel, etc.)
Monthly Financial Trends

🛠 Tools & Skills Used
Microsoft Excel
Pivot Tables
Charts & Graphs
Data Analysis
Financial Modeling
